# 7GUIs_java
1. Counter -> This is an GUI with Java Swing where by clicking the Click Me" button it increases the count by 1 per click and shows the clicking number in a textbox.

2. Modern Counter -> I just tried to add some extra features in this program, which can actually increase or decrease the count and also can reset the count by just one click.

3. Temperature Converter -> This is actually a normal temperature converter, where you can give input in celcius and the output will be in fahrenheit automatically.
